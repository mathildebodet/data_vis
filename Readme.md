This project explores a central question:
What really shapes students’ eating habits?
More specifically, it analyzes how three key constraints — budget, time, and academic workload — influence students’ food behaviors and attitudes.
The goal is to move beyond common assumptions and provide data-driven insights into how students adapt their eating habits under real-life constraints.

The code is in the project.py file, it runs using streamlit with the following command "streamlit run project.py".
The requirements.txt has all the libraries you need to download.
